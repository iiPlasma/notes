# PreUnit
## Preterito
### Reasons For Use
1. Defined Time Frame
2. These Cases:
    - Vivid action
    - Completed event
    - Beginning or end of an action
    - Sequence of events
3. Trigger Words:
    - Anoche
    - Ayer
    - Anteayer
    - El `__` pasado
    - De repente
    - De pronto
    - Una vez

## Imperfect
### Reasons For Use
1. Undefined Time Frame
2. These Cases:
    - Description
    - Age or time
    - Weather
    - Habits, Used To...
    - Was/Were --ing
3. Trigger Words:
    - Mientras
    - Generalmental
    - De vez en cuando
    - A veces
    - Frecuentemente
